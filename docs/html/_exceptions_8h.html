Подробное описание

Иерархия пользовательских исключений приложения.

Все ошибки между функциональными блоками программы (хранилище, шифрование, сессии) и ошибки, предназначенные для показа пользователю, передаются через этот механизм исключений и обрабатываются на границе обработчиков команд бота.
