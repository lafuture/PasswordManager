Подробное описание

Хранит производные ключи шифрования пользователей в оперативной памяти.

Ключ выводится из загруженного пользователем файла-ключа и никогда не сохраняется на диск. Каждая сессия имеет тайм-аут бездействия: если в течение заданного времени не было обращений, ключ автоматически удаляется. Класс потокобезопасен, так как обращения происходят из потока опроса Telegram.
